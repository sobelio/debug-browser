  <files>src/browser/session.rs</files>
  <action>
Add convenience methods to the Browser struct that wrap chromiumoxide's Page API. These are the building blocks all future commands will use.

Study agent-browser's `src/browser.ts` for the pattern — it wraps Playwright's page with methods like navigateTo(), evaluateScript(), addInitScript(). We do the same with chromiumoxide's Page.

Methods to add on Browser (or on a new PageHandle wrapper):

1. navigate(&self, url: &str) -> Result&lt;()&gt;
   - Get or create a page
   - page.goto(url).await
   - Wait for navigation to complete

2. evaluate(&self, expression: &str) -> Result&lt;serde_json::Value&gt;
   - page.evaluate_expression(expression).await
   - Return the result as serde_json::Value

3. add_init_script(&self, script: &str) -> Result&lt;()&gt;
   - page.evaluate_on_new_document(script).await
   - This is critical for Phase 6+7 (React DevTools hook injection)

4. execute_cdp&lt;T: chromiumoxide_types::Command&gt;(&self, cmd: T) -> Result&lt;T::Response&gt;
   - page.execute(cmd).await
   - Generic method for any typed CDP command

The Browser should hold an Option&lt;Page&gt; for the active page, lazily creating it on first use. Reference agent-browser's BrowserManager which tracks currentPage and creates pages on demand.

Add tracing::debug! calls for all CDP operations (like agent-browser logs to stderr).

Test 1: Browser launch and version check
```rust
#[tokio::test]
async fn test_browser_launch_and_version() {
    let browser = Browser::launch(true).await.unwrap(); // headless=true
    let page = browser.new_page("about:blank").await.unwrap();

    // Execute Browser.getVersion CDP command
    use chromiumoxide::cdp::browser_protocol::browser::GetVersionParams;
    let version = page.execute(GetVersionParams::default()).await.unwrap();
    assert!(version.product.contains("Chrome") || version.product.contains("Headless"));

    browser.close().await.unwrap();
}
```

Test 2: Navigation and evaluate
```rust
#[tokio::test]
async fn test_navigate_and_evaluate() {
    let browser = Browser::launch(true).await.unwrap();
    browser.navigate("about:blank").await.unwrap();

    let result = browser.evaluate("1 + 1").await.unwrap();
    assert_eq!(result, serde_json::json!(2));

    browser.close().await.unwrap();
}
```

Test 3: Console event listener (proves event streaming works)
```rust
#[tokio::test]
async fn test_console_events() {
    use chromiumoxide::cdp::js_protocol::runtime::EventConsoleApiCalled;
    use futures_util::StreamExt;

    let browser = Browser::launch(true).await.unwrap();
    let page = browser.new_page("about:blank").await.unwrap();

    let mut console_events = page.event_listener::<EventConsoleApiCalled>().await.unwrap();

    // Trigger a console.log
    page.evaluate_expression("console.log('hello from test')").await.unwrap();

    // Wait for the event (with timeout)
    let event = tokio::time::timeout(
        std::time::Duration::from_secs(5),
        console_events.next()
    ).await.unwrap().unwrap();

    assert_eq!(event.r#type, chromiumoxide::cdp::js_protocol::runtime::ConsoleApiCalledType::Log);

    browser.close().await.unwrap();
}
```
