The BrowserManager should be simplified to:
- launch(options) — launches Playwright chromium
- close() — closes browser
- navigateTo(url, waitUntil?) — navigates current page
- evaluateScript(script, args?) — evaluates JS on page
- addInitScript(script) — adds script to run on every navigation
- getConsoleMessages(clear?) — returns captured console logs
- getPageErrors(clear?) — returns captured errors
- getCurrentUrl() — returns current page URL
- getTitle() — returns current page title
- getPage() — returns the Playwright Page for direct use by actions
- isLaunched() — returns whether browser is running
