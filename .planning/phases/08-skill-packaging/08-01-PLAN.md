Create `scripts/install.sh` — a portable shell script that builds and installs debug-browser.

The script should:
1. Accept `--prefix` argument (default: `$HOME/.local`)
2. Build the Rust CLI: `cargo build --release`
3. Build the daemon: `cd daemon && npm install && npm run build`
4. Install to standard layout:
   - `$PREFIX/bin/debug-browser` (copy binary)
   - `$PREFIX/share/debug-browser/daemon/dist/` (copy daemon dist/)
   - `$PREFIX/share/debug-browser/daemon/node_modules/` (copy production node_modules only — use `npm ci --omit=dev` in a temp step or just copy node_modules excluding devDeps)
   - `$PREFIX/share/debug-browser/daemon/package.json` (needed by node for module resolution)
   - `$PREFIX/share/debug-browser/daemon/dist/scripts/` (the injected JS scripts)
5. Print instructions: "Add $PREFIX/bin to PATH" and "export DEBUG_BROWSER_HOME=$PREFIX/share/debug-browser (optional — auto-discovered from binary location)"

Use `#!/usr/bin/env bash`, set -euo pipefail. Use `install -Dm755` for binary, `cp -r` for daemon files. Keep it simple — no fancy argument parsing beyond --prefix.

Also create `scripts/uninstall.sh` that removes everything from the prefix.
