- CookiesClearCommand: action 'cookies_clear'

**Daemon protocol.ts:** Add Zod schemas:
- cookiesGetSchema: urls optional string array
- cookiesSetSchema: cookies array with all properties validated

**Daemon actions.ts:** Add handlers:
- handleCookiesGet: `context.cookies(command.urls)` — return cookie array

- `cookies clear` → cookies_clear

**Rust commands.rs:** Add builder functions cookies_get, cookies_set, cookies_clear.
