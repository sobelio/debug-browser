
2. Create GitHub release:
```bash
gh release create v1.1.0 --title "v1.1.0: Shipping" --notes "$(cat <<'NOTES'
## What's New

- **Skill Packaging**: Claude Code skill with proper daemon discovery and standalone install script
- **Nix Flake**: `nix profile install github:sobelio/debug-browser` — builds Rust CLI + Node.js daemon with Playwright
- **Public Repo**: Clean README with install/usage instructions

## Install

\`\`\`bash
# Nix (recommended)
nix profile install github:sobelio/debug-browser

# Manual
cargo build --release
cd daemon && npm install && npm run build
\`\`\`

## Full Changelog

Built on v1.0 MVP (React component tree inspection, hook state, console capture, script injection, automated testing, context efficiency).
NOTES
)"
```

3. Verify release:
```bash
gh release view v1.1.0
```
