**Header:** debug-browser — React debugging via headless browser, optimized for AI agents

**What it does:** 2-3 sentences. Rust CLI that controls a headless Chromium browser via a Node.js daemon (Playwright). Designed for LLM/AI agent workflows — all output is structured text, no TUI. Core value: React component tree inspection with props, state, and hooks.

**Quick Start:**
```bash

nix profile install github:sobelio/debug-browser

# Manual
cargo build --release
cd daemon && npm install && npm run build
```

**Usage Examples:** Show 3-4 key commands:
```bash
debug-browser navigate https://example.com
debug-browser react-tree
debug-browser react-hooks MyComponent
debug-browser screenshot
```

**Architecture:** One paragraph + diagram showing: Rust CLI → Unix socket → Node.js daemon → Playwright → Chrome

**Commands:** Table of all commands with one-line descriptions. Get the full list from the CLI help or from skill/SKILL.md.

**Development:**
```bash
nix develop  # or: cargo build && cd daemon && npm install
cargo test
```

**License:** MIT

Keep it concise — this is a developer tool README, not a marketing page. No badges, no emojis.
